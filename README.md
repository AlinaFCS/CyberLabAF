

# CyberLab

This repository contains hands-on cybersecurity labs and investigations focused on entry-level SOC (Security Operations Center) analyst skills.

The purpose of this lab is to document practical analysis of security events such as phishing emails, suspicious indicators, and basic incident investigations using industry-relevant tools and methodologies.

---

## Areas of Focus

- Phishing email analysis
- Email header investigation
- IOC (Indicators of Compromise) extraction and enrichment
- Basic SIEM-style alert investigation
- CyberChef decoding and data analysis
- Blue Team fundamentals

---

## Repository Structure

CyberLab/
├── phishing/
│    ├── case-01/
│    │    └── analysis.md
├── siem/
├── cyberchef/
└── README.md
Each case includes:
- A short case summary
- Technical analysis
- Identified IOCs (defanged)
- Tools used
- Final verdict and reasoning
